# Halo Training

Halo Training is a fitness apparel storefront built with Astro and React. It leans into bold typography, sportswear branding, and editorial landing-page composition.

## Highlights

- High-contrast performance brand aesthetic
- Category-based storefront structure
- Astro pages with light React-powered interactions

## Stack

- Astro
- React
- Tailwind CSS

## Quick start

```bash
bun install
bun run dev
```

Checks and build:

```bash
bun run check
bun run build
```
